Filipino to baybayin.py

Utilizing the unicode of the baybayin scripts as output

Hyphenator.py

Dividing word into syllable that matches the baybayin scripts

speech to baybayin.py

Using speech as an input and outputs a word transliterated to baybayin


Image generator using DCGAN.ipynb

GAN model to generate images of the baybayin scripts to be use in displaying script images

Filipino to baybayin Image Generate.ipynb

Combination of hyphenator, Image generator using DCGAN, and displaying output baybayin word using images generated in the DCGAN model
